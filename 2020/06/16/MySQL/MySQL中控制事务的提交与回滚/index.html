一、如何控制事务 - COMMIT / ROLLBACK

在 MySQL 中，事务的自动提交状态默认是开启的。

二、总结

  1. 自动提交

    • 查看自动提交状态：SELECT @@AUTOCOMMIT ；

    • 设置自动提交状态：SET AUTOCOMMIT = 0 。

  2. 手动提交

    @@AUTOCOMMIT = 0 时，使用 COMMIT 命令提交事务。

  3. 事务回滚

    @@AUTOCOMMIT = 0 时，使用 ROLLBACK 命令回滚事务。
