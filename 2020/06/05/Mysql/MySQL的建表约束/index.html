Primary Key与Unique Key区别

一、作为Primary Key的域/域组不能为null，而Unique Key可以。

二、在一个表中只能有一个Primary Key，而多个Unique Key可以同时存在。

更大的区别在逻辑设计上。Primary Key一般在逻辑设计中用作记录标识，这也是设置Primary Key的本来用意，而Unique Key只是为了保证域/域组的唯一性。
